Latar Belakang

Program Fun Run pertama SMA Negeri 1 Singosari telah diselenggarakan dengan sukses, hal tersebut membuka peluang baru bagi SMANESI untuk menyelenggarakan kompetisi Fun Run dengan jangkauan partisipan yang lebih luas. Pada tahun 2024 ini OSIS SMA Negeri 1 Singosari menyelenggarakan kembali kompetisi Fun Run dengan skala yang lebih luas yakni melibatkan SMP/MTS/Sederajat, SMA/SMK/MA/Sederajat dan masyarakat umum dengan batas usia tertentu.

Kegiatan Fun Run ini diselenggarakan dengan tujuan meningkatkan kepedulian terhadap pentingnya kesehatan dan memupuk sportivitas antar peserta. Selain itu, tujuan utama dari diadakannya kegiatan ini adalah untuk membangun hubungan yang harmonis, dan menjalin silaturahmi dengan siswa/i aktif, sekaligus peserta dari seluruh wilayah Malang Raya. Kegiatan Fun Run ini memiliki tema yakni “Fobsicalls” (Face Obstacles, Find Challenges, and Get Success) yang memiliki arti “Hadapi Rintangan, Temukan Tantangan, dan Dapatkan Keberhasilan”. Pengambilan tema tersebut didasari dengan diadakannya kegiatan Fun Run kali ini yang dapat membangkitkan semangat peserta dalam menghadapi rintangan dan tantangan.

Frequently Asked Questions

FAQ SMANESI FUN RUN 2024

Ada berapa kategori jenjang perlombaan pada SMANESI Fun Run 2024?

di dalam SMANESI FUN RUN 2024 terdapat kategori umum, kategori SMA/MA/Sederajat, kategori SMP/MTS/Sederajat, dan kategori Para Siswa SMANESI.

Berapa jarak rute yang digunakan pada SMANESI Fun Run 2024?

Total jarak rute untuk 3 kategori yakni 8 km.

Apakah ada COT (Cut Off Time)?

Untuk jarak rute 8Km = 1 jam atau 1 jam 30 menit

Pukul berapa acara ini dimulai?

Acara ini akan dibuka mulai pukul 5 pagi dan akan dimulai pukul 7 pagi.

Apa saja yang saya dapatkan dalam acara ini?

peserta akan mendapatkan air minum, medali, dan nomer dada.

Berapa biaya pendaftaran untuk mengikuti SMANESI Fun Run 2024?

untuk umum = 75k
untuk SMA/MA/Sederajat = 60k
untuk SMP/MTS/Sederajat =50k
untuk Siswa SMANESI = 45k

Apabila saya telah terdaftar namun berhalangan mengikuti event, apakah bisa mendapatkan kembali uang pendaftaran?

Peserta SMANESI Fun Run bisa mendapatkan kembali uang pendaftaran dengan catatan ada halangan terkait kesehatan tubuh peserta (sakit dengan disertai surat dokter) pada tanggal pelaksanaan atau pra pelaksanaan.

Bagaimana saya dapat mengetahui bahwa saya sukses terdaftar?

Peserta akan dikonfirmasi sukses terdaftar melalui email yang akan terkirim kepada calon peserta.

Bagaimana cara mendaftar SMANESI Fun Run 2024?

Tata cara pendaftaran ada pada juknis pendaftaran yang sudah tertera.

Apakah pendaftaran saya bisa diberikan atau dipindahtangankan ke orang lain?

Pendaftaran calon peserta tidak dapat dipindahkan kepada orang lain tanpa pengecualian.

Apabila saya memiliki riwayat penyakit tertentu, apakah saya bisa mengikuti SMANESI FUN Run 2024?

Diharapkan untuk para calon peserta dapat mengetahui kemampuan daya tahan tubuh masing masing sebelum mengikuti kegiatan.

Bagaimana jika saya melakukan kesalahan dalam pengisian data dan telah terkirim?

Peserta dapat konfirmasi dengan cara menghubungi pihak narahubung yang tertera (Rinjani: 0896-8236-1197 atau Dimas: 0878-5087-9049). Untuk jarak rute 8Km = 1 jam atau 1 jam 30 menit

Apakah saya diwajibkan untuk mengunggah foto kartu pelajar/kartu identitas sebagai syarat pendaftaran?

Seluruh calon peserta diwajibkan mengunggah foto kartu pelajar/kartu identitas guna mempermudah proses verifikasi data oleh panitia penyelenggara. Seluruh data yang diunggah bersifat rahasia dan tidak akan disebarluaskan dengan alasan apapun dan keamanannya dapat dipertanggungjawabkan oleh panitia penyelenggara.

Dalam bentuk apakah kartu identitas yang harus saya unggah sebagai syarat pendaftaran?

Kartu identitas tersebut dapat berupa Kartu Tanda Penduduk (KTP) terkini dengan data yang akurat dan dilarang melakukan rekayasa data dalam bentuk apapun.
